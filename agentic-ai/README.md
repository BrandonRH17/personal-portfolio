## Project Summaries

### 1. Tasker — AI Sales Agent

Built for the **Microsoft AI Agents Hackathon 2025** (Team Neutrino). Tasker is an AI-powered sales assistant that transforms how high-ticket real estate teams handle leads — responding within 5 minutes, generating personalized quotes in under 30 seconds, and creating custom prospect videos in under 3 minutes. Uses a single-agent architecture with OpenAPI tool-use to orchestrate vector search, quote generation, and video creation through Azure Logic Apps.

**Stack:** Azure AI Agents, Azure Logic Apps, Azure AI Search, Azure SQL, GPT-4o-mini, text-embedding-3-large, FastAPI, BannerBear, Retool
**Highlights:** Single-agent with 3 tool integrations, semantic vector search across clients/properties/projects, automated CRM management, personalized video generation per lead persona
