# ICU Patient Prioritization - Executive Presentations

This folder contains executive-level presentations and reports derived from the MIMIC-IV Healthcare Analytics project, focused on ICU patient prioritization and clinical decision support.

## Available Formats

### Executive Summary Report
- **Markdown:** `Executive_Summary_ICU_Patient_Prioritization.md` — editable text format
- **HTML:** `Executive_Summary_ICU_Patient_Prioritization.html` — web-ready, recommended for PDF export
- **Jupyter Notebook:** `Executive_Presentation_ICU_Prioritization.ipynb` — technical version with code and regenerable visualizations

## How to Export as PDF

### From HTML (Recommended)
1. Open `Executive_Summary_ICU_Patient_Prioritization.html` in your browser
2. Press `Ctrl + P` (Windows/Linux) or `Cmd + P` (Mac)
3. Select "Save as PDF" or "Microsoft Print to PDF"
4. Click Save

### From Markdown
1. Edit `Executive_Summary_ICU_Patient_Prioritization.md` as needed
2. Use a markdown editor with PDF export (e.g., Typora, VS Code with extensions)
3. Export to PDF

## Report Contents

The executive summary covers:
- Critical patient risk factors and age-based risk stratification
- High-mortality diagnoses and comorbidity patterns
- ICU resource allocation insights
- Actionable patient prioritization matrix
- KPIs and clinical decision algorithms
- Length of stay analysis

**Data Source:** MIMIC-IV Clinical Database Demo v2.2 (100 patients, 275 admissions, 140 ICU stays)
